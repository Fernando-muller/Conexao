# Integração Java com MySQL 📦
# Descrição

Este projeto foi desenvolvido como atividade do curso do Senac, com o objetivo de praticar a integração entre Java e MySQL.

A aplicação demonstra como uma aplicação Java pode se conectar a um banco de dados, realizar operações básicas e manipular informações armazenadas no MySQL.

O projeto foi criado com foco no aprendizado de persistência de dados, conexão com banco de dados e organização de código em aplicações Java.

# Funcionalidades

Conexão da aplicação Java com o banco de dados MySQL

Inserção de dados no banco

Consulta de informações armazenadas

Prática de persistência de dados
